THIS IS A GUIDE TO INSTALL THIS PARTICULAR PROJECT ON THE MACHINE.

1. Import the SQL Schema present in the Assignment folder.
2. The tables along with the values should be created and added to the database. Please make sure the database name is smartportables.
3. Extract the Project Zip file to the webapps folder under tomcat.
4. Open the compiler and set up all the required PATH and CLASSPATH variables: MySQL, MongoDB, Gson, Bson, JDK.
5. Compile all the java classes in the same terminal/cmd.
6. Startup the tomcat server in the same terminal/cmd.
7. Open http://localhost:8080/Project
